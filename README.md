# AutoFlowLayout
自定义LinearLayout，支持自动换行，指定行数,实现流式布局

Step 1. Add the JitPack repository to your build file
```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
Step 2. Add the dependency
```
dependencies {
	        compile 'com.github.LRH1993:AutoFlowLayout:1.0.2'
	}
```
